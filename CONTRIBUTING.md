# Contributing to Lost Things Search

Thank you for your interest in contributing to this project! Please read this information before submitting a Pull Request

## Branches

This repository has two branches:

| Branch | Description                                                                                                    |
| ------ | -------------------------------------------------------------------------------------------------------------- |
| `dev`  | Main development branch. **All Pull Requests must target this branch**                                         |
| `main` | Release branch. Updated only by repository owner when publishing a new version. Only accepts merges from `dev` |

## Committing

Please follow Conventional Commits in messages

## Local Setup
```bash
git clone https://github.com/laptop-coder/lost-things-search.git
cd lost-things-search
git checkout dev
make dev
```

For additional info see [README.md](https://github.com/laptop-coder/lost-things-search/blob/main/README.md)
